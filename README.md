# Password Manager

## Requirements

- Python 3.12

## Setup

On linux:
'''console
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
'''

On windows:
'''console
python -m venv .venv
.venv\Scripts\activate.bat
pip install -r requirements.txt
'''

## Running

'''console
python main.py
'''

or

'''console
make run
'''
