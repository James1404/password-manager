# Password Manager

A password manager for CS1032, by James Barnfather.

<!-- Getting Started--!>

## Prerequisites

* Python 3.12

### Installation

#### Linux
Use:
```console
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Or:
```console
make init
```

#### Windows
```console
python -m venv .venv
.venv\Scripts\activate.bat
pip install -r requirements.txt
```

## Running

```console
python main.py
```

or

```console
make run
```
