<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

* Python 3.12

### Installation

#### Linux
Use:
```sh
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Or:
```sh
make init
```

#### Windows
```sh
python -m venv .venv
.venv\Scripts\activate.bat
pip install -r requirements.txt
```

<!-- USAGE -->
## Usage
