# Password Manager

## Requirements

- Python 3.12

## Setup

On linux:
''
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
''

On windows:
''
python -m venv .venv
.venv\Scripts\activate.bat
pip install -r requirements.txt
''

## Running

''
python main.py
''

or

''
make run
''
